# MOTDcraft

The ultimate Minecraft Java Edition MOTD creator. 
MOTDcraft is a simple, single-file web application that lets you instantly build, preview, and export server MOTDs with full support for vanilla formatting codes.

## Features

- **Live Pixel-Perfect Preview**: See exactly how your MOTD looks in the Minecraft server list menu, featuring authentic fonts, colors, and layout.
- **Full Vanilla Formatting**: Support for all 16 Minecraft colors (`§0` - `§f`) and text formats (bold, italic, underline, strikethrough, reset).
- **Realistic Obfuscation**: Authentic `§k` scramble animation directly in your browser.
- **Customizable Server Plate**: Upload your own 64x64 server icon and modify the server name.
- **Two-Line Support**: Native support for two-line MOTDs.
- **Gradient Generator**: Easily generate Minecraft-compatible linear color gradients between two colors.
- **Multiple Output Formats**: One-click copying for Raw (`§`), JSON Text Component (`{"text":...}`), and MiniMessage (`<color>...`) formats.
- **No Dependencies**: Everything is bundled into a single zero-dependency HTML file.

## Usage

Simply open `index.html` in any modern web browser to start using the tool.
